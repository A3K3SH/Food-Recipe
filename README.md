# Fork & Fire - Recipe App

A modern, feature-rich recipe discovery application built with React, featuring beautiful animations, smart search, and favorites management powered by TheMealDB API.

## Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Steps
1. Clone the repository
```bash
   https://github.com/A3K3SH/Food-Recipe.git
```

2. Navigate to the project directory
```bash
   cd Food-Recipe-App
```

3. Install dependencies
```bash
   npm install
   # or
   yarn install
```

4. Start the development server
```bash
   npm run dev
   # or
   yarn dev
```

5. Open your browser and visit
```
   http://localhost:5173
```
